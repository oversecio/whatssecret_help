Per-App Configuration

Note: This configuration page can also be opened by tapping the WhatsSecret config button.

Options

General options

General Settings

Panic when screen off

Enter panic mode when screen turns off. Enable this if you want to enter Panic Mode every time the phone screen turns off. You will have to dial a secrete code in the dialer to reactivate WhatsSecret. Use this to make you phone look inconspicuous every tine youturn on the screen.

Hide launcher icon in Panic Mode

Enable this to hide the WhatsSecret launcher icon while in panic mode. You will have to dial a secrete code in the dialer to reactivate WhatsSecret and unhide the launcher icon.

Secret Dialer Code

Configure the code to dial (in the dialer app) to reactivate WhatsSecret (exit panic mode)

Allow Screenshots

By default, WhatsSecret prevents the system from taking screenshots while showing sensitive information. Depending on the way the screenshots are made, taking screenshots either fails or produces a blank or black image.

You can disable this behavior temporarily if you want to take screenshots to show or tell about WhatsSecret, but we strongly recommend to keep this setting OFF!

Config Button

Shows the Config Button that allows direct access to this config page.

Start hidden

When checked, WhatsSecret will not initially be shown when opening Whatsapp. You will have to tap on the (invisible) [Hide Button] (/buttons#button_hide) to show the WhatsSecret screen overlay.

Hide Button

Shows the [Hide Button] (/buttons#button_hide) that allows to temporarily hide the WhatsSecret screen overlay.

Hint: This buttons also doubles as Panic Button, so it's recommended to always keep this button enabled!

Decrypt/Clear Button

Shows the Decrypt Button that allows to decrypt or clear the current input field.

Visual

Visual appearance

Fine-Tune the visual appearance of the WhatsSecret screen overlays - things like fonts, sizes, padding, colors, transparencies etc.
