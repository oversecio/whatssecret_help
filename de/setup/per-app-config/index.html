Allgemeine Optionen

Allgemeine Einstellungen

Panik wenn Bildschirm aus

Aktiviert den Panik-Modus wenn der Bildschirm aus geht. Benutze dies, damit dein Handy unverdächtig aussieht, nachdem du den Bildschirm wieder anschaltest.

WhatsSecret Icon verstecken im Panik Modus

Versteckt das WhatsSecret Start Icon im Launcher, wenn der Panik Modus aktiv ist

Geheimer Wähl Code

Wähle diesen Code in deiner Telefon app, um WhatsSecret nach einer Panik zu reaktivieren und das Launcher Icon wieder anzuzeigen.

Screenshot erlauben

Standardmässig verhindert WhatsSecret, dass Screenshots gemacht werden während potentiell private (entschlüsselte) Texte und Schlüssel angezeigt werden. Entweder funktioneren die Screenshots überhaupt nicht, oder sie sind komplett schwarz.

Du kannst diesen Schutz (zeitweise) ausschalten, wenn du z.B. Screenshots erstellen möchtest, um über WhatsSecret zu berichten. Bitte vergesse nicht, die Funktion danach wieder einzuschalten.

Konfigurations-Button

Zeigt den Konfigurations-Button an der dich Dich direkt zu diesr Seite bringt. .

Versteckt starten

Wenn aktiviert, wird WhatsSecret nicht sofort angezeigt, wenn du zu Whatsapp wechselst. Du musst dann zuerst den (unsichtbaren!) Verstecken Button um WhatsSecret wieder anzuzeigen

Verstecken Button

zeigt den [Verstecken Button] (/buttons#button_hide) an, mit dem du WhatsSecret zeitweise verstekcen kannst.

Tipp: Dieser Button dient auch als Panik Button, er sollte deshalb immer aktiviert sein!

Entschlüsseln- / Löschen Button

Zeigt den Entschlüsseln Button an mit dem Du das Eingabefeld entschlüsseln oder löschen kannst..
