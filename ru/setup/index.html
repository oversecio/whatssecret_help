Необходимая настройка

The basic operating principle of WhatsSecret is that it constantly monitors the screen for encrypted text.

To be able to read the screen contents, it has to be enabled as a so called "Accessibility Service".

So as the first step, before you can use WhatsSecret, you need to enable WhatsSecret as an Accessibility Service in Androids "Accessibility Settings".

Tapping on "Configure" will lead you directly to the relevant Settings page:

Основной принцип работы WhatsSecret заключается в том, что он постоянно контролирует экран для зашифрованного текста.

Чтобы иметь возможность читать содержимое экрана, его нужно включить как так называемую «службу специальных возможностей».

Итак, на первом этапе, прежде чем вы сможете использовать WhatsSecret, вам нужно включить WhatsSecret в качестве Службы специальных возможностей в «Настройках специальных возможностей» Андроидов.

Нажатие кнопки «Настроить» приведет вас непосредственно к соответствующей странице настроек:

Теперь нажмите «WhatsSecret»



Теперь нажмите на переключатель в правом верхнем углу



Нажмите «Вкл».



Теперь переключатель должен быть включен. Нажмите «НАЗАД»,



Теперь WhatsSecret должен быть включен. Нажмите «НАЗАД»,



На некоторых телефонах с пользовательской версией Android страница настроек, которая позволяет настраивать службы специальных возможностей, недоступна или недоступна в ожидаемом месте.

Попробуйте вручную найти страницу «Доступность» в настройках телефона, а затем включить WhatsSecret как службу специальных возможностей .

Общие настройки

Паника при отключении экрана

Войдите в режим паники, когда экран выключится. Включите это, если вы хотите входить в режим паники каждый раз, когда экран телефона выключается. Чтобы активировать WhatsSecret, вам придется набирать секретный код в модуле набора номера. Используйте это, чтобы заставить вас выглядеть непринужденно на каждом экране.

Скрыть значок запуска в режиме паники

Включите это, чтобы скрыть значок запуска WhatsSecret в режиме паники. Вам нужно будет набрать секретный код в дозвоне, чтобы повторно активировать WhatsSecret и отобразить значок запуска.

Код секретного кода

Настройте код для набора (в приложении дозвона), чтобы повторно активировать WhatsSecret (выход из режима паники)

Разрешить скриншоты

По умолчанию WhatsSecret запрещает системе снимать скриншоты, показывая конфиденциальную информацию. В зависимости от способа создания скриншотов снятие скриншотов либо сбой, либо получение пустого или черного изображения.

Вы можете временно отключить это поведение, если вы хотите делать скриншоты, чтобы показать или рассказать об WhatsSecret, но мы настоятельно рекомендуем не отключать этот параметр!

Кнопка конфигурации

Показывает кнопку Кнопка «Настройка», которая позволяет получить прямой доступ к этой странице конфигурации.

Запустить скрыто

Если флажок установлен, WhatsSecret первоначально не будет отображаться при входе в Whatsapp;. Вам нужно будет нажать на невидимую кнопку Кнопка «Скрыть», чтобы отобразить наложение экрана WhatsSecret.

Кнопка «Скрыть»

Показывает кнопку Кнопка «Скрыть», которая позволяет временно скрыть накладку экрана WhatsSecret.

Подсказка: эти кнопки также дублируются как кнопка паники, поэтому рекомендуется всегда держать эту кнопку включенной!

Кнопка «Расшифровать / Очистить»

Показывает кнопку Кнопка «Расшифровать / Очистить», которая позволяет расшифровать или очистить текущее поле ввода.
