Botões

Botão Ocultar

O botão Ocultar permite ocultar temporariamente a sobreposição do appname;.

Toque neste botão para ocultar temporariamente todas as sobreposições appname;.

Toque novamente na mesma posição (enquanto este e todos os outros botões estão ocultos) para ocultar!

Este botão também funciona como um botão de pânico: Toque-o longamente para entrar no Modo pânico

Dê um toque duplo neste botão traz a página de configuração por-app onde você pode configurar vários aspectos de como o appname; funciona em conjunto com o aplicativo atual.

Nota: Este botão pode ser arrastado e colocado em qualquer lugar na tela.

Botão Criptografar

Toque neste botão para encriptar o texto introduzido. Depois que o texto foi criptografado, este botão desaparece automaticamente. Portanto, se você colocar esse botão sobre o aplicativo subjacente "enviar" ou "salvar" botão, você pode ter certeza que você não acidentalmente enviar ou salvar texto sem formatação.

Se você ainda não criptografou nada neste aplicativo, tocando neste botão, aparecerá uma tela para inserir ou modificar os parâmetros de criptografia. Da próxima vez que você tocar neste botão, esses parâmetros serão utilizados ea criptografia acontecerá instantaneamente. Se você ainda quiser alterar os parâmetros de criptografia, toque longamente neste botão para exibir com força a tela de criptografia.

  • Dica: Dê um duplo toque neste botão irá escondê-lo por um curto período de tempo, para que os botões do aplicativo subjacente podem ser aproveitados. Portanto, se o Botão Criptografar estiver posicionado bem acima do botão de envio do aplicativo subjacente, você pode triplo tocar nessa posição para enviar uma mensagem não criptografado!

Nota: Este botão pode ser arrastado e colocado em torno do campo de entrada.

Botão Descriptografar / Limpar

Este botão muda seu ícone e função dependendo do texto no campo de entrada:

Há texto criptografado no campo de entrada. Toque neste botão para descriptografar e colocar o texto sem formatação novamente no campo de entrada.

Nota: Esta funcionalidade não está disponível no Android Kitkat por razões de segurança!

Há algum texto no campo de entrada. Tocar neste botão limpa todo o texto.

Nota: Este botão pode ser arrastado e colocado em torno do campo de entrada.

Botão Informações

Toque neste botão para entrar no modo info.

Enquanto no modo info, haverá uma versão menor deste botão ao lado de cada texto criptografado encontrado na tela. Toque no botão info pequeno para mostrar informações detalhadas de criptografia sobre este texto.

Enquanto no modo de informação, este botão mostra como acima. Toque para sair do modo info.

Nota: Este botão pode ser arrastado e colocado em qualquer lugar na tela.

Nota: Este botão não está habilitado por padrão. Clique no botão de configuração para abrir a página de configuração por-app onde você pode ativar o botão info.

*Note: Long-tapping the Info-Button brings up the per-app configuration page (see below). You can thus enable the info button and disable the config button in the per-app configuration page and still have access to the app config by long-tapping the info button.

  • Nota: O botão de Info abre a página de configuração por-app (veja abaixo). Você pode, portanto, habilitar o botão info e desativar o botão de configuração na página de configuração por-app e ainda ter acesso à configuração do app por longas tocando no botão info.

Botão Config.

Ao aceder a este botão é apresentada a página de configuração por-app onde pode configurar vários aspectos de como o appname; funciona em conjunto com a aplicação actual.

Nota: Este botão pode ser arrastado e colocado em qualquer lugar na tela.
