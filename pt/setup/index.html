Configuração

Configuração necessária

O princípio de funcionamento básico de WhatsSecret é que ele monitora constantemente a tela para texto criptografado.

Para poder ler o conteúdo da tela, ele deve ser habilitado como o chamado "Serviço de Acessibilidade".

Assim, como primeiro passo, antes de poder utilizar WhatsSecret, tem de activar WhatsSecret como um Serviço de Acessibilidade nas configurações "Acessibilidade" do Android.

Toque em "Configurar" irá levá-lo diretamente para a página de configurações relevantes:

Agora toque em "WhatsSecret"



Agora toque no botão alternar no canto superior direito



Toque em "OK"



Agora, o interruptor deve estar LIGADO. Toque no botão Voltar.



Agora WhatsSecret deve estar "ligado". Toque no botão Voltar.



Em alguns telefones com uma versão personalizada do Android, a página de configurações que permite configurar os Serviços de acessibilidade não está disponível ou não no local esperado.

Tente manualmente localizar a página "Acessibilidade" nas Definições do seu telefone e, em seguida, activar WhatsSecret como um Serviço de Acessibilidade.

Configurações Gerais

Pânico quando tela desligada

Entre no modo de pânico quando a tela é desligada. Active esta opção se pretender introduzir o Modo de pânico sempre que o ecrã do telefone for desligado. Você precisará discar um código secreto no discador para reativar WhatsSecret. Use isso para fazer você olhar telefone inconspicuous cada tino youturn na tela.

Ocultar o ícone do iniciador no modo de pânico

Habilite isso para ocultar o ícone do lançador WhatsSecret enquanto estiver no modo de pânico. Terá de marcar um código secreto no marcador para reactivar WhatsSecret e mostrar o ícone do lançador.

Código do discador secreto

Configure o código para discar (no aplicativo de discagem) para ativar o Overseas (sair do modo de pânico)

Permitir capturas de ecrã

Por padrão, o WhatsSecret impede que o sistema faça capturas de tela enquanto exibe informações confidenciais. Dependendo da maneira como as capturas de tela são feitas, fazer capturas de tela falhar ou produzir uma imagem em branco ou preto.

Você pode desativar esse comportamento temporariamente se você quiser tirar screenshots para mostrar ou falar sobre WhatsSecret, mas é altamente recomendável manter essa configuração desligado!

Iniciar oculto

Quando marcado, WhatsSecret não será inicialmente mostrado ao inserir Whatsapp. Você terá que tocar no (invisível)Botão Ocultar para mostrar a sobreposição de tela WhatsSecret.

Mostra o Botão Ocultar que permite ocultar temporariamente a sobreposição de tela WhatsSecret.

Dica: Este botão também funciona como [Botão Pânico] (/panic), por isso recomenda-se manter sempre este botão ativado!

Conteúdo de Chamariz

Ao usar [Codificação Invisível] (/codificação), para enganar qualquer observador de terceiros, você pode automaticamente adicionar texto de chamariz visível no lugar do texto criptografado invisível.

Sob a guia "Chamariz" das configurações principais do aplicativo, você pode definir seus próprios padrões de chamariz ou modificar os padrões padrão.

Dica: Aqui estão alguns grandes sites onde você pode gerar conteúdo chamariz de

  • Letras de músicas
  • Pan Ipsum
